## Usage

When a commit's changes are primarily generated by a coding agent, add a `Co-Authored-By` footer naming the agent:

```
Co-Authored-By: Claude Sonnet 4.6 <noreply@anthropic.com>
```

This is the **only** indicator of AI involvement that should appear in the commit. Do not add phrases like `Generated by AI`, `Written with Claude`, or similar markers in the subject, body, or anywhere else.

### Example

```
fix: handle null pointer in user serializer

Closes #88
Co-Authored-By: Claude Sonnet 4.6 <noreply@anthropic.com>
```

## Key Points

- Applies whenever the agent authored most of the diff, even if a human reviewed or lightly edited it
- Use the agent's actual model name with a `noreply@`-style email — see [features-footers](features-footers.md) for general token format rules
- Multiple agents or co-authors stack as separate `Co-Authored-By` lines
- Keep the subject and body free of AI markers — attribution lives only in the `Co-Authored-By` footer
